# 🎟️ Rifa Digital

Projeto acadêmico da disciplina **Processo e Requisitos de Software (PRS)**.

## Objetivo

Desenvolver um sistema web simples para venda de rifas digitais para arrecadação de fundos.

## Contexto

O sistema permite que organizadores criem rifas online e que usuários comprem números de forma simples e segura.

## Usuários

- Organizador da rifa
- Comprador
- Vendedor

## Funcionalidades

- Criar campanha de rifa
- Exibir números disponíveis
- Comprar número
- Registrar comprador
- Realizar sorteio

## Artefatos de Engenharia de Software

Este repositório prioriza a documentação do projeto:

- Visão do produto
- Requisitos
- Personas
- Arquitetura
- Modelo de dados
- Backlog ágil
- Roadmap
