# Stakeholders do Projeto --- Rifa Digital

Stakeholders são indivíduos ou organizações que possuem interesse no
sistema.

------------------------------------------------------------------------

## Organizador

Interesse: - criar rifas - acompanhar vendas - registrar pagamentos

Impacto: - principal usuário do sistema

------------------------------------------------------------------------

## Participantes

Interesse: - visualizar números disponíveis - escolher números -
acompanhar sorteio

Impacto: - usuários finais do sistema

------------------------------------------------------------------------

## Instituição Organizadora

Interesse: - arrecadar recursos - transparência na gestão da rifa

Impacto: - beneficiário da campanha

------------------------------------------------------------------------

## Desenvolvedores

Interesse: - implementar funcionalidades - garantir qualidade do sistema

Impacto: - responsáveis pelo desenvolvimento
