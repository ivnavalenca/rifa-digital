
# Engineering Portal

Bem-vindo ao **Portal de Engenharia do Sistema Rifa Digital**.

Este portal centraliza toda a documentação técnica do sistema, permitindo navegar pelas diferentes áreas da engenharia de software do projeto.

---

## Visão Geral da Engenharia

O projeto **Rifa Digital** foi estruturado utilizando boas práticas de engenharia de software, contemplando:

- Engenharia de produto
- Experiência do usuário (UX)
- Engenharia de requisitos
- Arquitetura de software
- Arquitetura de dados
- Engenharia de testes
- Qualidade de software
- DevOps e CI/CD
- Observabilidade da engenharia

## Sobre o Portal

Este portal foi desenvolvido para fornecer uma visão integrada da engenharia do sistema, permitindo compreender:

- como os requisitos se conectam à arquitetura
- como os testes validam as funcionalidades
- como os dados suportam o sistema
- como a qualidade é monitorada
