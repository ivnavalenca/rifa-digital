# Casos de Uso --- Rifa Digital

## Atores

-   Organizador
-   Participante

------------------------------------------------------------------------

## Caso de Uso 1: Criar Rifa

Ator: Organizador

Fluxo Principal: 1. Organizador acessa o sistema. 2. Seleciona opção de
criar rifa. 3. Informa nome, descrição, quantidade de números e valor.
4. Define data do sorteio. 5. Sistema registra a nova rifa.

Resultado: campanha de rifa criada.

------------------------------------------------------------------------

## Caso de Uso 2: Escolher Número

Ator: Participante

Fluxo Principal: 1. Participante acessa a página da rifa. 2. Sistema
exibe números disponíveis. 3. Participante seleciona um número. 4.
Sistema solicita dados do participante. 5. Sistema registra a reserva do
número.

Resultado: número reservado.

------------------------------------------------------------------------

## Caso de Uso 3: Registrar Pagamento

Ator: Organizador

Fluxo Principal: 1. Organizador acessa lista de números reservados. 2.
Seleciona número. 3. Registra pagamento. 4. Sistema marca número como
vendido.

Resultado: pagamento confirmado.

------------------------------------------------------------------------

## Caso de Uso 4: Realizar Sorteio

Ator: Organizador

Fluxo Principal: 1. Organizador acessa a rifa. 2. Seleciona opção
realizar sorteio. 3. Sistema seleciona número vencedor. 4. Sistema
registra resultado.

Resultado: vencedor definido.
