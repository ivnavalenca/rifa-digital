# Visão do Produto

## Nome do sistema
Rifa Digital

## Descrição

Sistema web para gerenciamento e venda de números de rifa, permitindo que organizadores criem campanhas e participantes comprem números para participar do sorteio.

## Usuários

- Organizador
- Participante

## Funcionalidades principais

- Criar campanha de rifa
- Escolher número
- Registrar comprador
- Registrar pagamento
- Realizar sorteio
