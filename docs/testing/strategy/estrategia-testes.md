# Estratégia de Testes --- Rifa Digital

## Objetivo

Definir a abordagem de verificação e validação do sistema Rifa Digital
para garantir que os requisitos definidos sejam corretamente
implementados.

------------------------------------------------------------------------

## Tipos de Teste

### Teste de Requisitos

Verificar se cada requisito funcional e não funcional está implementado
corretamente.

### Teste Funcional

Validar as funcionalidades principais do sistema:

-   criar rifa
-   escolher número
-   registrar pagamento
-   realizar sorteio

### Teste de Regras de Negócio

Garantir que regras importantes sejam respeitadas:

-   número não pode ser selecionado duas vezes
-   pagamento deve ser confirmado antes do sorteio
-   sorteio só ocorre após venda dos números

### Teste de Aceitação

Validar se o sistema atende às necessidades dos usuários definidos nas
personas.

------------------------------------------------------------------------

## Critérios de Qualidade

-   integridade dos dados
-   consistência dos requisitos
-   confiabilidade do sistema
-   usabilidade da interface

------------------------------------------------------------------------

## Ferramentas

Os testes podem ser realizados utilizando:

-   testes manuais
-   checklist de validação
-   revisão de requisitos
