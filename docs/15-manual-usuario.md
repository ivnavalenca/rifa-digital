# Manual do Usuário --- Rifa Digital

## 1. Introdução

O sistema **Rifa Digital** permite criar e gerenciar campanhas de rifa
de forma simples e organizada. Ele possibilita que organizadores criem
rifas e que participantes escolham números para concorrer a prêmios.

Este manual descreve como utilizar as principais funcionalidades do
sistema.

------------------------------------------------------------------------

## 2. Acessando o Sistema

1.  Acesse a página inicial do sistema.
2.  Visualize a campanha de rifa disponível.
3.  Escolha a ação desejada conforme seu perfil de usuário.

Perfis principais:

-   **Organizador**: responsável por criar e gerenciar a rifa.
-   **Participante**: usuário que escolhe números e participa do
    sorteio.

------------------------------------------------------------------------

## 3. Criar uma Rifa (Organizador)

Passos para criar uma nova rifa:

1.  Acessar o sistema.
2.  Selecionar a opção **Criar Rifa**.
3.  Informar os seguintes dados:
    -   nome da rifa
    -   descrição
    -   quantidade de números
    -   valor de cada número
    -   data do sorteio
4.  Confirmar criação da rifa.

Resultado esperado:

A campanha de rifa será criada e estará disponível para os
participantes.

------------------------------------------------------------------------

## 4. Visualizar Números Disponíveis

1.  Acessar a página da rifa.
2.  O sistema exibirá todos os números.
3.  Os números serão apresentados com status:

-   disponível
-   reservado
-   vendido

------------------------------------------------------------------------

## 5. Escolher um Número (Participante)

Passos para participar da rifa:

1.  Acessar a página da rifa.
2.  Visualizar os números disponíveis.
3.  Selecionar um número.
4.  Informar dados do participante:
    -   nome
    -   telefone
    -   email (opcional)
5.  Confirmar a reserva do número.

Resultado esperado:

O número ficará reservado para o participante.

------------------------------------------------------------------------

## 6. Registrar Pagamento (Organizador)

1.  Acessar a lista de números reservados.
2.  Selecionar o número.
3.  Registrar confirmação de pagamento.
4.  O sistema marcará o número como **vendido**.

------------------------------------------------------------------------

## 7. Realizar Sorteio

Quando a venda dos números estiver concluída:

1.  Acessar a campanha da rifa.
2.  Selecionar **Realizar Sorteio**.
3.  O sistema selecionará um número vencedor.
4.  O resultado será exibido.

------------------------------------------------------------------------

## 8. Visualizar Resultado

Após o sorteio:

1.  Acessar a página da rifa.
2.  Visualizar o número vencedor.
3.  Verificar os dados do participante vencedor.

------------------------------------------------------------------------

## 9. Perguntas Frequentes

**Um número pode ser escolhido por mais de uma pessoa?**\
Não. O sistema impede a duplicação de números.

**O que acontece se o pagamento não for confirmado?**\
O número pode voltar a ficar disponível.

**Quem pode realizar o sorteio?**\
Apenas o organizador da rifa.

------------------------------------------------------------------------

## 10. Suporte

Em caso de dúvidas ou problemas no uso do sistema, entre em contato com
o organizador da campanha.
