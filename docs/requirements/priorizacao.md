# Priorização de Requisitos --- Método MoSCoW

Este documento apresenta a priorização dos requisitos do sistema
utilizando o método **MoSCoW**.

## Categorias

-   Must Have: essencial
-   Should Have: importante
-   Could Have: desejável
-   Won't Have: fora do escopo

------------------------------------------------------------------------
```
  Requisito                        Prioridade
  -------------------------------- ------------
  Criar campanha de rifa           Must
  Visualizar informações da rifa   Must
  Visualizar números disponíveis   Must
  Escolher número da rifa          Must
  Registrar comprador              Must
  Registrar pagamento              Should
  Visualizar progresso da rifa     Should
  Realizar sorteio                 Must
  Divulgar resultado               Should
  Compartilhar rifa                Could
  Visualizar histórico da rifa     Could
```
