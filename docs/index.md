## Sobre o Projeto

O **Rifa Digital** é um sistema desenvolvido para permitir a criação e gestão de rifas digitais.

Funcionalidades principais:

- criação de campanhas
- seleção de números
- reserva de números
- confirmação de pagamento
- sorteio automático
